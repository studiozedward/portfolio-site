Pip-Token

VS Code extension · 2026

Pip-Token is a free, open-source VS Code extension that tracks Claude Code token usage in real time, dressed in a Pip-Boy aesthetic borrowed from the Fallout series. The status bar displays your current session's token spend, daily totals, and rate-limit headroom — without leaving your editor.

Built for developers who live inside Claude Code and want a glanceable view of their usage without context-switching to a dashboard.

Features

- Real-time session view: input/output splits, peak/off-peak counters, burn rate, and time-to-limit estimates

- Context window tracker showing how much of Claude's working memory is in use

- Cache hygiene monitor that warns before your prompt cache expires and shows how much it has saved

- Interactive chart tooltips with peak/off-peak/limit hit breakdown on hover

- Personalised limit thresholds learned from your own usage history

- Cost estimates in your local currency, calculated from API list pricing

- Contextual advisories that surface based on your current data

- Status bar pinned to every page showing peak status, context fill, burn rate, and week-to-date cost

- CRT flicker transitions and blip sound effects

- Onboarding flow to configure plan tier, currency, and timezone

Installation

Available on the VS Code Marketplace. Search for "Pip-Token" or install via the command palette.
