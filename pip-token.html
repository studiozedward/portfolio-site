Pip-Token

VS Code extension · 2026

[ project image ]

Pip-Token is a free, open-source VS Code extension that tracks Claude Code token usage in real time, dressed in a Pip-Boy aesthetic borrowed from the Fallout series. The status bar displays your current session's token spend, daily totals, and rate-limit headroom — without leaving your editor.

Built for developers who live inside Claude Code and want a glanceable view of their usage without context-switching to a dashboard.
